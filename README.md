# StudioSetting
 - 导入settings.jar，方便设置代码风格
 - 编写Android studio插件教程： 
 http://mp.weixin.qq.com/s/tW8AA4mRasL3UjRMuWB4rQ
 
 
### 用法：
Android Studio 依次选File--->Import Settings , 然后选择根目录下的settings.jar包导入即可
